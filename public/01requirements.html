Systémové požiadavky

Web Bus Simulator je určený pre spúšťanie na zariadeniach s inštalovanými webovými prehliadačmi Mozilla Firefox alebo Google Chrome a Safari s plnou podporou grafickej knižnice WebGL.

Kvalitu služieb a funkcionalitu ovládania pri používaní ostatných webových prehliadačov negarantujeme. 

Pokiaľ vaše zariadenie disponuje vyššie spomínaným webovým prehliadačom, overte si podporu knižnice WebGL kliknutím na nasledujúci odkaz.

Pokiaľ vaše zariadenie nedisponuje vyššie spomínaným webovým prehliadačom, môžete využiť nasledujúce odkazy pre potrebnú inštaláciu.
V prípade akýchkoľvek problémov odporúčame inštalovať aktuálne verzie :

PC verzie :

  • Mozilla Firefox
  • Google Chrome

Smartfónové verzie :

  • Mozilla Firefox
  • Google Chrome
  • Safari

Minimálne sytémové požiadavky na hardvérový výkon PC a smartfóny: 

  • Webový prehliadač : Mozilla Firefox, Google Chrome, Safari ( iOS).
  • Operačný systém: Windows 7 a vyššie, Distribúcie Linuxu a Unixu s podporou vyššie spomenutých webových prehliadačov, iOS 8 a vyššie.
  • Operačná pamäť RAM : 2GB
  • Procesor CPU : 1.6 GHz
  • Grafické procesory s podporou OpenGL.
    • Pre smartfóny s minimálnou taktovacou frekvenciou 500 MHz a vyššie.
    • Pre osobné počítače a notebooky s minimálnou taktovacou frekvenciou 675 MHz a operačnou pamäťou o veľkosti 512MB.
